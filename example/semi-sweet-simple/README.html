1. You must have Leiningen installed to follow these instructions. It's easy to install. Start here.

  2. To download and install everything you need to run the example, including Midje, type this: 

         % lein deps

    The downloaded jar files are put in the lib/ subdirectory.

  3. The file test/semi_sweet_simple/core_test.clj is intended to be good documentation of the different ways you can use midje.semi-sweet. You can run the tests like this: 

         % lein test

    Keep in mind that some of the "tests" intentionally fail.

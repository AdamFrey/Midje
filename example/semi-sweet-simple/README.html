1. You must have Leiningen installed to follow these instructions. It's easy to install. Start here. 

  2.      % lein deps

    This will download and install everything you need to run the example, including Midje. It puts those things in the lib/ directory. 

  3.      % lein test

    This runs the tests in test/semi_sweet_simple/core_test.clj. You can look at the tests there.
